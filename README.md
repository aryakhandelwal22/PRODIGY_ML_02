# PRODIGY_ML_02  
## Customer Segmentation using K-Means Clustering

### Internship Task
This project is completed as part of the Machine Learning Internship at Prodigy InfoTech.

## Objective
To create a K-Means clustering algorithm to group retail store customers based on their purchase history.

The segmentation is performed using:

- Annual Income
- Spending Score

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning Concept Used

### K-Means Clustering
K-Means is an unsupervised machine learning algorithm used to group similar data points into clusters.

### Elbow Method
Used to determine the optimal number of clusters.

## Steps Performed

1. Loaded customer dataset
2. Selected relevant features
3. Applied Elbow Method
4. Implemented K-Means clustering
5. Visualized customer segments
6. Identified cluster centroids


## Output
The algorithm segmented customers into **5 clusters** based on income and spending behavior.
